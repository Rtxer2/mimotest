# i18n Implementation Plan

**Goal:** Add Chinese/English/Spanish language support to all frontend pages.

**Architecture:** react-i18next with JSON translation files, localStorage language preference, Select dropdown in header.

---

### Task 1: Install deps + i18n config + translation files
- Install react-i18next, i18next, i18next-browser-languagedetector
- Create `frontend/src/i18n/index.ts` config
- Create `frontend/src/i18n/locales/zh.json`, `en.json`, `es.json` with ALL translation keys
- Update `frontend/src/main.tsx` to import i18n

### Task 2: Update Layout, Sidebar, Login
- Add language selector in Layout header
- Update ConfigProvider with antd locale
- Replace all hardcoded text in Sidebar, Layout, Login, ProtectedRoute

### Task 3: Update Dashboard + AnalyticsDashboard
- Replace hardcoded text with t() calls

### Task 4: Update all other pages
- customers, orders, production, inventory, quality, approvals, notifications, system pages
- NotificationBell component

### Task 5: Final verification
- TypeScript check + build
